Db ​

数据存储管理相关的 API 接口。

作用 ​

轻松管理本地数据，读写方便。

initDb ​

初始化数据库。

ts
import { initDb } from "electron-prokit";

initDb()
  .then((res) => {
    console.error(res);
  })
  .catch((err) => {
    console.error(err);
  });

  .catch((err) => {
    console.error(err);
  });

💬 参数

ts
initDb = (file = defaultFile): Promise<boolean>
initDb = (file = defaultFile): Promise<boolean>
  • file：数据存储文件位置，默认为join(app.getPath("userData"), "db.json")。

writeDb ​

写数据。

ts
import { writeDb } from "electron-prokit";

writeDb("test", { name: "Hello", age: 11 });
import { writeDb } from "electron-prokit";

writeDb("test", { name: "Hello", age: 11 });

💬 参数

ts
writeDb = async (key: string, data: any): Promise<void>
writeDb = async (key: string, data: any): Promise<void>
  • key：数据存储 key
  • data：数据存储 data
