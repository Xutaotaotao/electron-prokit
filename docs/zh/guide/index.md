# 简介

本指南假设你熟悉 [Electron](https://www.electronjs.org/) 和 [Vite](https://vitejs.dev/)，熟悉之后阅读本指南更加容易理解框架的设计思想。
:::

## 解决什么样的问题

Electron开发在社区中有很多的解决方案，比较了一些方案，大概有这些共同的问题：

- 繁琐的搭建Electron项目的步骤
- 无法自主选择前端框架
- 各个进程本地开发代码热更新问题
- 构建打包一体化配置问题
- 渲染进程调用无法抹平差异问题
- 缺少成套的解决方案

electron-prokit旨在解决这些问题，让你拥有极致的桌面开发体验。
